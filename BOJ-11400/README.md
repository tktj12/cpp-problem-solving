# Problem
* https://www.acmicpc.net/problem/11400
* 그래프의 단절선 찾기
* 난이도 : 플레티넘 4

# Solution

### 방법 1 : DFS, 간선 분류
* 풀이
  * DFS로 단절선을 찾는다.
  * 단전설일 조건
	1. 본인과 자식을 잇는 간선 외에 본인한테 다시 올 수 있는 간선이 있으면 안 된다.
	즉, 자식을 제외한 자손 중에 역방향 간선이 있으면 안 된다.
  * 여기서 자식은 아직 DFS로 방문되지 않은 인접 정점이다.


* 시간 복잡도 :
  * $O(|V| + |E|)$
