# Problem
* https://www.acmicpc.net/problem/12094
* 2048 게임의 한 상태에서 5번 더 진행해서 얻을 수 있는 최대 점수 구하기
* 난이도 : 플레티넘 4

# Solution

### 방법 1 : 시뮬레이션, 구현, 가지치기
* 풀이
  * 한 상태에서 갈 수 있는 다음 상태는 최대 4가지이다.
  * 브루트포스로 10번 했을 때 갈 수 있는 모든 상태를 살펴본다.
  * 이때 현재 게임판의 최댓값이 남은 횟수 안에 아무리 커져도 현재 best 값 보다 크지 않을 경우, 혹은 
  게임판의 바뀐 상태가 이전 상태랑 같은 경우에는 더 이상 시뮬레이션 해보지 않는다.

* 시간 복잡도 :
  * $O(4^10N^2)$
