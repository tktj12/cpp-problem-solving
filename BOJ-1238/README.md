# Problem
* https://www.acmicpc.net/problem/1238
* 각 마을에서 X까지 왕복하는 데 가장 오랜 시간이 걸리는 마을 구하기
* 난이도 : 골드 3

# Solution

### 방법 1 : Floyd 모든 경로 최단 쌍
* 풀이
  * 플로이드 알고리즘으로 모든 최단 경로를 구해서 푼다.

* 시간 복잡도 :
  * $O(|V|^3)$
  * 아슬아슬하게 통과.
<br></br>

### 방법 2 : 다익스트라
* 풀이
  * X에서 다른 정점까지의 거리는 다익스트라 한번이면 $O(|E|\log |V|)$시간에 찾을 수 있다.
  * 문제는 다른 정점들로부터 X까지의 거리인데 이 문제에서는 정점이 1000개라서 그냥 구해도 되겠지만 
  더 최적화 하는 방법은 간선들을 방향을 반대로 돌리고 X를 기준으로 다익스트라 알고리즘을 수행하는 것이다. 
  이렇게 하면 다익스트라 2번 만에 문제를 풀 수 있다.
