# Problem
* https://www.acmicpc.net/problem/13172
* 페르마의 소정리 응용하기
* 난이도 : 골드 4

# Solution

### 방법 1 : 분할 정복
* 풀이
  * $a/b$꼴이 나올 때마다 모듈라 역원을 구한다.
	* $b$의 모듈라 역원을 구하는 공식 => $b^{-1} = b^{M-2} \mod M$
	* $M$과 $b$가 서로소일 때만 모듈라 역원이 존재하고, $M$이 소수일 때 위의 공식으로 모듈라 역원을 구할 수 있다.
  * $M$이 매우 크므로 분할 정복하여 거듭제곱을 수행한다.

* 시간 복잡도 :
  * $O(N\log M)$
