# Problem
* https://www.acmicpc.net/problem/13264
* 접미사 배열 구하기
* 난이도 : 플레티넘 3

# Solution

### 방법 1 : 맨버-마이어스(Manber-Myers) 접미사 배열 알고리즘
* 풀이
  * 맨버-마이어스 알고리즘은 $O(n\log^2 n)$ 시간 안에 접미사 배열을 사전순으로 구할 수 있다.

* 시간 복잡도 :
  * $O(n\log^2 n)$

* 입력 데이터 중에 정렬에 $O(n^2)$ 시간이 걸리는 데이터가 있는 것 같다. 
이 방법으로는 시간 초과가 난다.
