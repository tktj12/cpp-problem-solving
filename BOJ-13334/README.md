# Problem
* https://www.acmicpc.net/problem/13334
* 1차원 좌표에서 선분 L이 가장 많은 집-회사 선분을 포함하는 경우 구하기
* 난이도 : 골드 2

# Solution

### 방법 1 : 정렬, 큐
* 풀이
  * 집-사무실 선분을 오른쪽 좌표 순으로 정렬하고 순차적으로 방문한다.
  * 방문한 노드들을 왼쪽 좌표가 빠른 선분을 우선으로 하는 우선순위 큐에 저장하고 큐의 front가 L의 범위를 벗어났으면 항상 제거한다.
  * 큐의 top엔 항상 제거 1순위가 올라와 있다.

* 시간 복잡도 :
  * $O(n\log n)$
