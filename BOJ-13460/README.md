# Problem
* https://www.acmicpc.net/problem/13460
* 보드를 기울여서 빨간 구슬만 구멍으로 빼내기
* 난이도 : 골드 1

# Solution

### 방법 1 : 구현, 가지치키
* 풀이
  * 문제의 조건에 맞게 게임을 시뮬레이션 하는 것을 구현했다.
  * 파랑 구슬이 구멍으로 빠지는 경우, 게임 상태가 바뀌지 않는 경우와 이미 좋은 답이 있어서 더 이상 보지 않아도 되는 경우는 
  가지치기하였다.

* 시간 복잡도 :
  * $O(4^N)$
