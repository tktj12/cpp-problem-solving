# Problem
* https://www.acmicpc.net/problem/13549
* 1차원 좌표 N에서 규칙에 따라 K로 가장 빨리 가는 경우 구하기
* 난이도 : 골드 5

# Solution

### 방법 1 : 우선순위큐
* 풀이
  * 소모한 시간이 가장 적은 것부터 탐색한다.

* 시간 복잡도 :
  * $O(N)$
<br></br>

# Learning
* 재귀 호출로 메모이제이션을 할 때 다시 같은 곳을 탐색하는 경우 의도치 않은 예외가 발생할 수 있다.
* 이외에도 생각치 못한 예외가 많이 있었다. 문제를 풀 때 처음에는 일단 논리에 더 집중하는 게 좋은 것 같다. 
최적화는 그 다음이다.
* $O(|V|+|E|)$시간을 소요하는 0-1 BFS라는 알고리즘이 있단 걸 알게됐다. 
* 다익스트라는 $O(|E| \log |E|)$ 또는 $O(|E| \log |V|)$ 시간이 걸린다.
