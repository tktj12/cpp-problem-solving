# Problem
* https://www.acmicpc.net/problem/1504
* 1번 정점에서 두 정점 v1,v2을 거쳐 N으로 가는 최단 경로 구하기
* 난이도 : 골드 4

# Solution

### 방법 1 : 다익스트라
* 풀이
  * v1 정점에서 다른 정점으로의 최단거리, v2에서 다른 정점으로의 최단거리를 구해놓고 
  두 경로 (1-v1-v2-N), (1-v2-v1-N) 중 최단 거리를 고른다.

* 시간 복잡도 :
  * $O(|E|\log |V|)$
