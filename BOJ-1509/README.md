# Problem
* https://www.acmicpc.net/problem/1509
* 문자열을 팰린드롬인 문자열들로 분할하는 경우 중 최소로 분할하는 경우 구하기
* 난이도 : 골드 1

# Solution

### 방법 1 : Manacher, DP
* 풀이
  * $DP[i]$ : $i$번 인덱스까지 최소의 팰린드롬 분할 개수. 
  * 매내처 알고리즘으로 팰린드롬 부분 문자열들을 구하고, $DP$ 배열을 완성한다.
  * $i$번 인덱스를 중심으로 하는 팰린드롬의 반지름이 $p$이면 $0 \le k \le p$에 대해 다음을 구할 수 있다.
  $DP[i+k] = min(DP[i+k], DP[i-k-1] + 1)$

* 시간 복잡도 :
  * $O(N^2)$
