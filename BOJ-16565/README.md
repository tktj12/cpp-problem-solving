# Problem
* https://www.acmicpc.net/problem/16565
* 트럼프 카드에서 N개의 카드를 뽑았을 때 포카드를 만들 수 있는 확률 구하기
* 난이도 : 골드 2

# Solution
 naive하게는 $_{52}C_N$개의 조합을 살펴야 함.   
$N=4$일 때 $ans = 13$,   
$N=5 : ans = 13 * _{52-4}C_1$   
$N=6 : ans = 13 * _{52-4}C_2$   
$N=7 : ans = 13 * _{52-4}C_3$   
$N=8 : ans = 13 * _{52-4}C_4 - _{13}C_2$   
$N=9 : ans = 13*_{52-4}C_5 - _{13}C_2 * _{52-8}C_1$   
...
$N = 12 : ans = 13*_{52-4}C_8 - _{13}C_2 * _{52-8}C_4 + _{13}C_3$   
...
$N = 52 : ans = _{13}C_1*_{52-4}C_{N-4} - _{13}C_2 * _{52-8}C_{N-8} + ... - ... + ... - ... ... + _{13}C_{13} * _{4}C_{N-52}$

### 방법 1 : DP, 수학, 조합론, 포함 배제의 원리
* 풀이
  * $S_{N,i}$이 $N$개의 카드 중 포카드가 i개의 세트만 있는 모든 조합이라고 할 때,   
  포카드가 1개 이상 있는 조합 $T$는 $S_{N,1} \cup S_{N,2} \cup ... \cup S_{N,13}$ 이다. 
  이때 $T$의 크기 $|T|$는 포함 배제의 원리에 따라 아래와 같이 구할 수 있다.
  $$ans_N = \sum_{i=1}^{[N/4]} ((-1)^{i-1} * _{13}C_i* _{52 - 4 * i}C_{N - 4 * i})$$
  

* 시간 복잡도 :
  * $O(N^2)$
