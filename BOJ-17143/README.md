# Problem
* https://www.acmicpc.net/problem/17143
* 문제 조건대로 구현하기
* 난이도 : 골드 1

# Solution

### 방법 1 : 구현, 시뮬레이션
* 풀이
  * 이 문제는 상어의 움직임을 실수 없이 $O(1)$에 구현하는 게 관건이다.
  * 상어가 맵의 두 끝을 찍고 오면 다시 같은 상태에 있게 되는데 이 경우 지금까지 계산한 게 의미가 없으므로 
  마지막으로 두 끝을 찍고 온 후의 위치 변화만 빠르게 계산해준다. 
  * 위,아래,왼쪽,오른쪽 각각의 경우에 대해 방향이 바뀌지 않는 경우, 한번 바뀌는 경우, 두번 바뀌는 경우를 잘 생각해서 조건문으로 구현했다. 
  한 방향에 대해서만 잘 생각하면 다른 방향들도 어렵지 않게 생각할 수 있다.

* 시간 복잡도 :
  * $O(C(R+M))$, $R,C$ : 행,열 크기, $M$ : 상어 개수
