# Problem
* https://www.acmicpc.net/problem/17401
* 적혈구의 모든 이동경로의 개수 구하기
* 난이도 : 플레티넘 5

# Solution

### 방법 1 : 수학, 분할정복, 그래프
* 풀이
  * 인접 행렬로 그래프를 표현한다.
  * 행렬곱으로 적혈구의 이동경로 개수를 구할 수 있다.
	* '현재 이동경로 행렬' X '혈관 지도' = '다음 이동경로 행렬'
	* 문제의 정답은 이것을 D번 반복한 행렬이다.
	* '현재 이동경로 행렬'은 처음에 단위 행렬로 초기화한다.
	* '혈관 지도'는 매 초 바뀐다.
  * D가 매우 크므로 분할 정복을 이용하여 최적화한다.

* 시간 복잡도 :
  * $O(N^3(T + \log D))$
