# Problem
* https://www.acmicpc.net/problem/1766
* 문제 중에 우선순위가 있는 문제가 있을 때 최대한 작은 번호의 문제부터 푸는 방법 구하기
* 난이도 : 골드 2

# Solution

### 방법 1 : 우선순위 큐
* 풀이
  * 현재 풀 수 있는 문제를 우선순위 큐에 넣으면 현재 풀 수 있는 문제 중 가장 작은 번호부터 풀 수 있다.
  * 현재 풀 수 있는지 아닌지는 자기보다 우선인 문제들이 전부 나왔는지에 달렸는데, 
  자기보다 우선인 문제들의 개수를 $cntTurn$ 변수에 저장하여 카운트하고, 어떤 문제를 풀 때마다 
  그 문제를 우선으로 하고 있는 문제들의 목록을 $children$ 배열에 저장하여 빠르게 접근할 수 있게 하였다.

* 시간 복잡도 :
  * $O(M + N\log N)$
