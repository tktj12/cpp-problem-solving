# Problem
* https://www.acmicpc.net/problem/18837
* 중복되는 숫자가 있을 때 K번째 LIS 구하기
* 난이도 : 다이아3

# Solution
18838번을 먼저 푼 후에 풀었다. 18838번 풀이 : https://velog.io/@tktj12/BOJ-18838   
18838번 풀이에서 추가로 중복되는 수가 나타나는 경우에 대한 예외처리를 해주어야 한다.
예를 들어 {1,3,6,3,5} 에서 문제의 정의에 따라 사전순으로 두 번째인 LIS는 1,3,6이 아닌 1,3,5이다. 
그리고 세 번째 LIS가 1,3,6 이고, 4번째는 없다.
