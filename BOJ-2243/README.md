# Problem
* https://www.acmicpc.net/problem/2243
* 사탕의 맛에 따라 사탕을 넣고 빼고, 맛의 순위에 따라 사탕을 빼는 경우 어떤 사탕을 뺄지 구하기
* 난이도 : 플레티넘 5

# Solution

### 방법 1 : treap
* 풀이
  * 키에 따라 원소를 빠르게 탐색하고, 해당 원소가 몇 번째인지 빠르게 접근하기 위해 
대략적으로(roughly) 균형 이진 트리 자료 구조인 treap을 사용하여 문제를 해결한다.

* 시간 복잡도 :
  * $O(n\log m)$
