# Problem
* https://www.acmicpc.net/problem/2533
* 본인이 얼리아답터이거나 지인이 모두 얼리아답터여야 할 때 최소한으로 얼리아답터를 정하는 경우 구하기
* 난이도 : 골드 3

# Solution

### 방법 1 : DFS
* 풀이
  * 리프노드를 인접 정점으로 갖고 있는 노드는 무조건 얼리아답터여야 이득이다. 따라서 이러한 노드들부터 
  얼리아답터로 지정해나가면 문제를 풀 수 있을 것이다.
  * 따라서 DFS를 사용하는 게 가장 적절해보인다.

* 시간 복잡도 :
  * $O(|V| + |E|) = O(N)$
