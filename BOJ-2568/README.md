# Problem
* https://www.acmicpc.net/problem/2568
* 전깃줄이 교차하지 않게 최소 개수의 전깃줄 없애기
* 난이도 : 플레티넘 5

# Solution
 그리디하게 무조건 많이 겹치는 선부터 제거하는 방법이 가능할 것 같다. 
 그런데 이 방법을 빠른 시간 복잡도로 구현하는 게 어렵다.   

### 방법 1 : LIS (longest increasing subsequence)
* 풀이
  * 전봇대 A의 숫자가 수열의 순서이고 B의 숫자가 수열의 수라고 생각하면 LIS를 구함으로써 
  서로 교차하지 않는 가장 많은 전깃줄의 개수를 구할 수 있다. 따라서 이 과정에서 선택하지 않은 
  전깃줄을 구한다.
  * LIS를 구할 때 $i( > 0)$번째 수를 최신화한 순간에 $i-1$번째 수가 무엇이었는지를 기록해야한다. 
  이걸 기억해야 나중에 LIS를 끝에서부터 거꾸로 만들 수 있기 때문이다.

* 시간 복잡도 :
  * $O(N \log N)$
