# Problem
* https://www.acmicpc.net/problem/2638
* 외부와 두 변 이상 맞닿은 치즈만 녹을 때 모든 치즈가 녹는 데 걸리는 시간 구하기
* 난이도 : 골드3

# Solution

### 방법 1 : BFS
* 풀이
  * 외부공간의 공기가 BFS로 퍼진다고 생각하고 치즈와 두 번 이상 접촉하는지 확인한다. 
  두 번 이상 접촉하는 치즈를 없애고 모든 치즈가 없어질 때까지 반복한다.

* 시간 복잡도 :
  * $O(N*M*(N+M))$
<br></br>

# Learning
* 이 풀이는 한 번 치즈를 없앨 때마다 맵에 전부 공기를 퍼트렸는데, 굳이 그럴 필요 없이 없어진 치즈의 위치부터 다시 BFS를 하면 
$O(N*M)$ 시간 안에 풀 수 있다.
