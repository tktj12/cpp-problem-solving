# Problem
* https://www.acmicpc.net/problem/27172
* 게임의 규칙에 따라 진행했을 때 플레이어들의 최종 점수 구하기
* 난이도 : 골드 5

# Solution

### 방법 1 : 구현
* 풀이
  * 플레이어의 카드 숫자를 기준으로 오름차순으로 정렬하고 작은 수 부터 x2, x3, ... 해가며 나누어 떨어지는 수들을 계산한다.

* 시간 복잡도 :
  * $$O(\sum_{k=1}^n 1000000/k)$$
  * $1000000/k$인 이유는 x의 최댓값이 1000000이기 때문이다.
  * 최악의 경우 계산 횟수가 14,000,000 정도이다.
