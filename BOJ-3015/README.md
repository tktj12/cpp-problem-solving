# Problem
* https://www.acmicpc.net/problem/3015
* A와 B 사이에 A 또는 B보다 키가 큰 사람이 없는 (A,B) 쌍의 수 구하기
* 난이도 : 플레티넘 5

# Solution

### 방법 1 : 스택
* 풀이
  * 스택의 top에는 현재 사람과 (A,B) 쌍을 만들 수 있는 A의 키가 있다. A < B 라면 스택을 pop하고 A > B 라면 스택에 B를 push한다.
  A == B 인 경우는 나중에 설명하겠다. 이렇게 스택에 정리하면 어떤 사람이 몇 개의 쌍을 만들 수 있는지 $O(N)$만에 구할 수 있다. 
  * 스택은 또한 다음 사람의 키가 top에 저장된 키와 같은 사람이 얼마나 나왔는지를 count한다. 즉, A == B인 경우 cnt 값을 
  증가시킨다. 따라서 모든 경우의 (A,B)쌍을 구한다.

* 시간 복잡도 :
  * $O(N)$
