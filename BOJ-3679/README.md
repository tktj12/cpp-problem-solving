# Problem
* https://www.acmicpc.net/problem/3679
* 단순 다각형(다각형의 경계가 스스로를 교차하지 않는 다각형) 구하기
* 난이도 : 플레티넘 4

# Solution

### 방법 1 : 그레이엄 스캔
* 풀이
  * 최하단의 최좌단 점을 기준으로 그레이엄 스캔을 하고 점을 순서대로 이으면 한 변이 다른 변을 교차할 일이 없다.
  * 이 풀이에서 그레이엄 스캔을 할 때, 두 점을 이은 직선이 기준점을 지난다면 기준점과 더 거리가 가까운 점이 우선인 것으로 
  하였다. 그런데 이렇게 했을 때 중간에 있는 점들은 문제가 없지만, 마지막에 위치하는 점 중 여러개가 
  기준점을 통과하는 한 직선이 위에 있을 수 있는데 이 상황에선 예외처리를 해주어야 한다.

* 시간 복잡도 :
  * $O(n\log n)$
