# Problem
* https://www.acmicpc.net/problem/9359
* $A \le x \le B$이고 $N$과 서로소인 자연수 $x$의 개수 구하기
* 난이도 : 골드 1

# Solution

### 방법 1 : 수학, 포함 배제의 원리
* 풀이
  * $x$의 소인수와 $N$의 소인수 중 겹치는 게 하나라도 있다면 서로소가 아니다. 
  따라서 $A$와 $B$ 사이에 $N$과 소인수가 겹치는 수의 개수를 센다. 
  * $N$의 소인수의 집합을 $S$라고 하자. 또한 $D(n, p) =$ 1부터 $n$까지 소수 $p$로 나누어 떨어지는 수의 집합이라고 하자.
  정답이 되는 집합은 다음과 같다. $(D(B,S_1) - D(A-1,S_1)) \cup (D(B,S_2) - D(A-1,S_2)) \cup ... \cup (D(B,S_{end}) - D(A-1,S_{end}))$   
  따라서 이 집합의 원소의 개수가 곧 정답이고, 이를 포함 배제의 원리를 이용해서 구할 수 있다.


* 시간 복잡도 :
  * $O(\sqrt{B})$
