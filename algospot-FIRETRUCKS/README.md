# Problem
* https://algospot.com/judge/problem/read/FIRETRUCKS
* 불이난 위치, 소방서 위치, 그리고 도로를 이용하는 시간이 주어질 때 가장 가까운 소방서에서 화재 장소로 이동하기 위해 걸리는 시간의 합 구하기
* 난이도 : 중

# Solution

### 방법 1 : 다익스트라 (종만북 풀이)
* 풀이
  * 다익스트라 알고리즘에서 처음에 우선순위 큐에 출발점을 넣을 때 소방서의 위치를 전부 넣고 시작하여
동시에 최단 거리를 구한다.

* 시간 복잡도 :
  * $O(|E|\log |V|)$
