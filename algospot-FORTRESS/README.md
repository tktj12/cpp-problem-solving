# Problem
* https://algospot.com/judge/problem/read/FORTRESS
* 성벽 간 이동이 제일 오래 걸리는 두 지점 구하기
* 난이도 : 중

# Solution

### 방법 1 : 트리 
* 풀이
  * 하나의 성벽이 노드이고 외벽을 루트 노드로 한 트리를 만든다.
  * 자식 노드는 부모 노드에 속하면서 형제 노드에게 속하지 않는 성벽들이다. 따라서 
자손 노드는 부모 노드 안에 속하는 모든 성벽이다.
  * 한 노드에서 가장 깊은 두 자식 노드의 깊이의 합이 답이 될 수 있다. 따라서 이 값 중 가장 큰 값을 구한다.

* 시간 복잡도 :
  * $O(n\log n)$
