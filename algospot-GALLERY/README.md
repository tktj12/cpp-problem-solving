# Problem
* https://algospot.com/judge/problem/read/GALLERY
* 인접한 갤러리(정점)를 감시할 수 있을 때 필요한 감시카메라의 최소 개수 구하기
* 난이도 : 중

# Solution
* 마지막 줄의 문제에 조건에 따라 미술관의 경로가 트리 형태라는 것을 알 수 있다.
* 리프노드의 부모노드는 무조건 감시 카메라를 설치하는 게 좋다.

### 방법 1 : 그리디
* 풀이
  * 감시카메라가 설치되지 않은 리프노드를 찾고 그 부모노드를 우선 감시카메라를 설치한다.
  * 이 과정을 더 이상 할 수 없을 때까지 반복한다.

* 시간 복잡도 :
  * $O(|V|^2)$

* 부모 노드를 marked로 처리하면 안 되는 상황이 있다. 결론적으로 틀린 알고리즘이다.
<br></br>

### 방법 2 : DFS
* 풀이
  * DFS로 탐색한 자식 노드들의 상태를 보고 감시카메라를 설치할지 판단하여 문제를 해결한다.
  * DFS로 탐색한다면 자식 노드들의 자손들은 모두 감시되고 있는 상황이라고 해석해도 되기 때문에 올바른 알고리즘이다.

* 시간 복잡도 :
  * $O(|V| + |E|)$
