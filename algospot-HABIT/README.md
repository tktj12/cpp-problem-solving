# Problem
* https://algospot.com/judge/problem/read/HABIT
* K번 이상 등장하는 가장 긴 부분 문자열 구하기
* 난이도 : 중

# Solution

### 방법 1 : 접미사 배열
* 풀이
  * 접미사 배열은 문자열의 접미사들을 사전 순으로 정렬한 배열이다. (정확히는 접미사가 시작되는 인덱스)
  * 문자열의 모든 부분 문자열은 접미사의 접두사로 구할 수 있다. ("abcd"의 접미사 "bcd"의 접두사 "bc"는 "abcd"의 부분 문자열)
  * 접미사 배열이 사전 순으로 정렬되어 있으므로 접미사 배열을 탐색하여 k번 반복되는 접두사가 무엇인지 알 수 있다.

* 시간 복잡도 :
  * 접미사 배열을 만드는 데 $O(n \log^2 n)$, 탐색하는 데 $O(n^2)$
  * 따라서 $O(n^2)$
