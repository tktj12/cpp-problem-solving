# Problem
* https://algospot.com/judge/problem/read/JLIS
* 가장 긴 합친 증가 부분 수열의 길이 구하기
* 난이도 : 하

# Solution

### 방법 1 : 동적 계획법
* 풀이
  * A수열의 a번째와 B수열의 b번째를 마지막으로 선택했을 때의 합친 증가 부분 수열의 최장 길이를 부분 문제로 생각
  * 따라서 a, b의 값을 기준으로 메모이제이션을 구현할 수 있다.
  * A, B의 원소는 1번부터 있고, 0번은 '선택 안함'을 의미한다.

* 시간 복잡도 :
  * $O(AB)$ - $A,B$의 길이에 비례
