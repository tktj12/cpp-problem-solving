# Problem
* https://algospot.com/judge/problem/read/MEETINGROOM
* 팀당 두 개의 회의 중 하나를 선택할 수 있을 때 모든 팀이 회의를 할 수 있는 경우 구하기
* 난이도 : 상상상

# Solution
* DP로는 $O(2^n)$ 이 걸린다.

### 방법 1 : 2-SAT
* 풀이
  * 회의별로 해당 회의를 하는 경우, 안 하는 경우의 두 가지로 나눌 수 있다. 이것은 Boolean 참 또는 거짓으로 표현할 수 있다. 
따라서 임의의 팀 $A$의 회의를 $A_1, A_2$ 라고 했을 때 $A_1, !A_1$으로 표현할 수 있다.
  * $A$는 $A_1, A_2$중 하나는 회의를 진행해야 한다. 따라서 $A_1 || A_2$ 이다.
  * 회의 $X$와 겹치는 회의 $Y$는 둘 중 하나는 무조건 회의를 진행해선 안 된다. 
따라서 $!X || !Y$ 이다.
  * 그러므로 이 문제는 2-SAT 문제이고 2-SAT을 해결하는 알고리즘으로 풀 수 있다.

* 시간 복잡도 :
  * $O(N^2)$
