# Problem
* https://algospot.com/judge/problem/read/MINASTIRITH
* 결국 공간 채우기
* 난이도 : 상

# Solution
* 초소가 감시하는 범위를 각도가 -$\pi ~ 3\pi$인 구간으로 변환
* 필요없는 구간(다른 구간에 포함되는)은 제거

### 방법 1 : 그리디, 구현
* 풀이
  * $1.$ 0을 포함하는 구간을 먼저 선택
  * $2.$ 이을 수 있는 구간 중 가장 긴 구간을 선택
  * $3.$ 처음 선택한 구간과 이어졌으면 종료
  * 가능한 모든 경우를 해보며 최소한으로 선택한 경우를 탐색
  * 0을 포함하는 구간이 없거나, 구간이 이어지지 않으면 불가능

* 시간 복잡도 :
  * $O(n^2)$

# Learning
* 함수 `atan(double)`는 `<cmath>` 헤더 파일에 정의되어있고 점 $P(x,y)$의 $y/x$ 값을 인수로 받으면 직선 $OP$와 
$x$축 양의 방향이 만드는 각도를 $\[ -\pi /2, \pi /2 \]$ 구간 내의 값으로 반환해준다
* 함수`atan2(y,x)`는 점 $P(x,y)$ 좌표와 원점을 있는 선분 $OP$와
$x$축 양의 방향이 만드는 각도를 $\[ -\pi, \pi \]$ 구간 내의 값으로 반환해준다
* 함수 `fmod(dividend, divisor)`는 실수 $dividend$를 실수 $divisor$로 나눈 나머지를 계산해준다.
