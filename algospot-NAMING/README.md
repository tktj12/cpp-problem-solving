# Problem
* https://algospot.com/judge/problem/read/NAMING
* 문자열의 접두사이면서 접미사인 부분 문자열들 구하기
* 난이도 : 중

# Solution

### 방법 1 : KMP 알고리즘 (커누스-모리스-프렛 알고리즘)
* 풀이
  * $getPartialMatch(str)$ 함수는 KMP 알고리즘을 구현한 것이다. 이 함수가 반환하는 배열 $p$의 요소
$p[i]$는 $str[..i]$의 접두사도 되고 접미사도 되는 문자열의 최대 길이이다. ($p[0]$은 구하지 않는다, $str[..i]$ : 길이가 $i+1$인 $str$의 부분 문자열)
  * 이 문제가 원하는 답인 "전체 문자열의 접두사도 되고 접미사도 되는 문자열들"을 길이 순으로 오름차순으로 정렬한
배열을 $answer$라고 할 때, 문자열 $answer[i]$는 $answer[i-1]$의 접두사이자 접미사일 수 밖에 없다.   
이것을 이용하여 문제를 해결한다.

* 시간 복잡도 :
  * $O(n)$, $n$ : 문자열의 길이
