# Problem
* https://algospot.com/judge/problem/read/NERDS
* 너드인 사람과 너드가 아닌 사람을 조건(이론)에 따라 구별할 수 있는 입력인지 판별하기
* 기하 문제
* 난이도 : 상

# Solution

### 방법 1 : 
* 풀이
  * 3차원 공간에서 $z=T$인 평면으로 두 다각형을 분리할 수 있다면 조건이 만족한다고 가정한다.
여기서 두 다각형은 너드인 벡터들이 만드는 다각형과 너드가 아닌 벡터들이 만드는 다각형이다.
  * 이때 $z = A*x + B *y$인 3차원 공간은 $x, y$평면에서 $y$축은 $x$축을 기준으로 반시계 방향으로 
$arctan(B)$ 만큼 기울고 $x$축은 $y$축을 기준으로 반시계 방향으로 $arctan(A)$만큼 기울었다고 볼 수 있다.
  * 따라서 $x,y$ 평면에서의 두 다각형을 한 직선으로 나눌 수 있으면 $A, B$ 값에 따라 $z=T$인 평면으로 두 다각형을 3차원 공간에서
나눌 수 있다고 볼 수 있다. (실제로 평면에 그림을 그려서 평면을 기울여보면 알 수 있다.)

* 시간 복잡도 :
  * $O(n^2)$
