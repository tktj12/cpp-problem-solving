# Problem
* https://algospot.com/judge/problem/read/NTHLON
* 두 선수 각각 운동 종목별 예상 기록이 주어질 때 두 선수가 비기게 종목을 선택하는 경우 중 최소한으로 시간을 소요하는 경우 구하기
* 난이도 : 중

# Solution

### 방법 1 : 다익스트라
* 풀이
  * 종목에서 A, B선수의 예상 기록의 차이를 하나의 정점으로 생각할 수 있다.
예를들어 (수영 : A - 2분 , B - 5분) 일 때 차이는 -3이고 -3이 정점이 될 수 있다.
  * 가중치의 경우 A선수와 B선수의 예상 기록의 합으로 생각할 수 있다.
  * 따라서 정점 0에서 시작해서 0으로 돌아오는 최단경로를 구하는 문제로 풀 수 있다.

* 시간 복잡도 :
  * 다익스트라 알고리즘의 시간 복잡도는 $O(|E|\log |V|)$ 이다. 
이 문제에서 $|E| = V*N$ 이다.
  * $O(|V|* N\log |V|)$
