# Problem
* https://algospot.com/judge/problem/read/NUMBERGAME
* 조합 게임에서 최대 몇 점 차이로 이기는지
* 난이도 : 중

# Solution
* 현우부터 시작
* 좌 우 끝에서 하나 가져가거나, 끝에서 두개 지우거나(두개 이상일 때)
* 모든 숫자가 없어졌을 때 끝

### 방법 1 : 완전 탐색
* 풀이
  * 한 턴에 가능한 4개의 경우에 대해 각각 재귀호출하여 최선을 선택

* 시간 복잡도 :
  * $O(4^n)$
<br></br>

### 방법 2 : 동적 계획법
* 결국 자기 차례일 때 게임판이 어떻게 배치되어있냐가 중요
* 게임판 배치를 최대한 단순화해야함

* 풀이
  * 게임판의 왼쪽 끝의 인덱스, 오른쪽 끝의 인덱스를 입력으로 받음. 예전 메모리가 있다면 반환

* 시간 복잡도 :
  * 부분 문제의 개수 $O(n^2)$ , 부분 문제를 푸는 데 $O(1)$ 시간이 걸리므로
  * $O(n^2)$
<br></br>
