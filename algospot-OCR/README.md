# Problem
* https://algospot.com/judge/problem/read/OCR
* 확률을 이용한 올바른 단어인식
* 난이도 : 상

# Solution

### 방법 1 : 완전 탐색, 재귀함수
* 풀이
  * (앞선 단어, 문장의 현재 분류 단어) 쌍을 입력으로 받고, 만들 수 있는 가장 높은 확률을 반환한다.
  * 앞서 가장 확률이 높은 단어라도 뒤쪽 단어들에 따라 전체 문장의 확률이 달라질 수 있기 때문에 모든 가능성을 탐색해봐야 한다.
  * 모든 단어를 탐색하기 때문에 함수 호출 한 번당 m개의 경우를 탐색한다.

* 시간 복잡도 :
  * $O(n^m)$
<br></br>

### 방법 2 : 동적 계획법
* 풀이
  * 방법 1에서 메모이제이션을 추가
  * 확률이 0에 너무 가까워지면 언더플로우가 날 수 있기 때문에 확률을 로그값으로 바꿔서 계산
  * 매 경우 최선의 답을 기록해놓고 마지막에 복원하여 답을 찾는 시간을 단축

* 시간 복잡도 :
  * $O(m^2n + mn) = O(m^2n)$
<br></br>
