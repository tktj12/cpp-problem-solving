# Problem
* https://algospot.com/judge/problem/read/PASS486
* [lo, hi] 범위 안에 n개의 약수를 가지는 숫자의 개수 구하기
* 난이도 : 중

# Solution

### 방법 1 : 에라토스테네스의 체
* 풀이
  * 에라토스테네스의 체를 이용, $1 \le n \le 10,000,000$인 모든 소수를 미리 찾고
$n$의 소인수 중 가장 작은 수를 미리 저장한다.
  * 소인수들의 개수로 약수의 개수를 구할 수 있는 것을 이용한다.   
예를 들어 24의 약수는 8개인데 24를 소인수 분해하면 $2^3 \times 3$이다.   
이때 24의 모든 약수들은 2와 3을 이용해서 만들 수 있다. 즉, (소인수 분해했을 때)2가 0개~3개 있는 약수,
3이 0개~1개 있는 약수가 있는 것이고 경우의 수로 계산하면 4x2 = 8개임을 알 수 있다.

* 시간 복잡도 :
  * $O(n)$과 비슷
