# Problem
* https://algospot.com/judge/problem/read/POTION
* 비율 맞추기
* 난이도 : 하

# Solution

### 방법 1 : 
* 접근
  * 현재 투입된 비율이 제일 높은 재료를 먼저 보는 게 좋을 것 같음

* 풀이
  * 현재 투입된 비율이 가장 큰 재료를 $s$, 이 재료에 총 넣는 양을 $k$라고 했을 때 모든 재료 $i$에 대해   
$k*r_i$가 $r_s$로 나누어 떨어져야 한다.

* 시간 복잡도 :
  * $O(n^2)$, $n$ : 재료의 양의 최댓값, $n$ 이하의 수 중 가장 차이가 큰 서로소가 있을 때가 $O(n^2)$이다.
