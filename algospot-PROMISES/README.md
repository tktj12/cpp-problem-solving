# Problem
* https://algospot.com/judge/problem/read/PROMISES
* 그래프 상에서 새로 추가하는 간선 중 의미 없는 간선의 개수 구하기
* 난이도 : 중

# Solution

### 방법 1 : 플로이드-워셜 모든 최단 경로 쌍 알고리즘
* 풀이
  * 이미 있는 도로들에 대해 미리 최단 경로들을 계산해 놓는다.
  * 도로가 새로 추가될 때 마다 최단 경로를 다시 계산한다. 
  이때 새로 추가된 도로인 u-v를 거치는 경로들만 계산하면 되기 때문에 $O(|V|^2)$ 시간이 소요된다.

* 시간 복잡도 :
  * $O(|V|^3 + |V|^2N)$
