# Problem
* https://algospot.com/judge/problem/read/QUANTIZE
* S개의 수로 분산이 최소가 되게 수열 양자화 하기
* 난이도 : 하

# Solution
* 수열을 정렬하여 풀 수 있다.

### 방법 1 : 동적 계획법
* 풀이
  * 수열을 정렬하고 구간별로 최소의 분산을 미리 계산해놓는다.
  * left를 시작으로하는 구간에서 s개의 수로 분산이 최소가 되게 양자화 하는 것을 부분문제로 생각
  * 이 부분 문제는 left, s 외에 영향을 받는 요소가 없기 때문에 최적 부분 구조가 성립하고,   
left와 s를 기준으로 메모이제이션을 할 수 있다.

* 시간 복잡도 :
  * 부분문제의 수 $O(NS)$와 각 부분 문제의 계산 시간 $O(N)$을 곱하면
  * $O(N^2 S)$
