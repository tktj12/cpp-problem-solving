# Problem
* https://algospot.com/judge/problem/read/SORTGAME
* 수열의 특정 구간을 뒤집는 연산이 있을 때 이 연산을 최소로 사용하여 수열을 정렬하는 경우 구하기
* 난이도 : 중

# Solution

### 방법 1 : BFS
* 풀이
  * N이 충분히 작다. 때문에 수열에 연산을 수행해서 얻을 수 있는 모든 결과에 대해 BFS로 탐색하여 
최단 거리를 찾는다.
  * N이 8 밖에 안 되기 때문에 N이 1인 경우부터 8인 경우까지 정답을 만드는 거리를 미리 계산해놓는다.

* 시간 복잡도 :
  * $O(N! \log N!)$
