# Problem
* https://algospot.com/judge/problem/read/TIMETRIP
* 웜홀을 통해 미래 또는 과거로 시간여행을 할 수 있을 때 지구에서 안드로메다로 가는 경로와 최소, 최대 시간 차 구하기
* 난이도 : 중

# Solution
* 안드로메다로 가는 경로 중에 사이클이 있을 경우
  * 사이클을 한 바퀴 돌았을 때 수가 증가하는 경우 양의 무한대로 발산할 수 있다.
  * 사이클을 한 바퀴 돌았을 때 수가 감소하는 경우 음의 무한대로 발산할 수 있다.

### 방법 1 : 벨만-포드 알고리즘 (Bellman-Ford)
* 풀이
  * 벨만-포드 알고리즘을 최단경로, 최대경로에 대해 적용한다.
  * 예외처리 : 벨만-포드 알고리즘을 |V|번 했을 때의 안드로메다까지의 최단(최장) 경로 값이 
음의(양의)무한대가 아니고 사이클을 발경했을 경우 
|V|번 더 완화를 시도하여 안드로메다까지의 최단(최장) 경로의 값이 바뀌었는지 확인한다.   
값이 바뀌지 않았다면 지구부터 안드로메다까지의 경로 상에서는 최단경로가 무한히 줄어들거나, 최대경로가 무한히 증가하는 
일이 없다는 의미이다.

* 시간 복잡도 :
  * $O(|V||E|)$
