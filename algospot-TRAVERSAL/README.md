# Problem
* https://algospot.com/judge/problem/read/TRAVERSAL
* 이진 트리를 전위, 중위 순회한 결과가 주어졌을 때 이 트리를 후위 순회한 결과를 출력
* 난이도 : 하

# Solution

### 방법 1 : 구현
* 풀이
  * 전위 순회, 중위 순회의 특징을 이용하였다.
  * 전위 순회한 결과, 중위 순회한 결과가 있을 때 해당 이진 트리의 루트가 될 수 있는 건 전위 순회한
결과의 맨 첫 번째이다.
  * 이 루트 노드가 중위 순회한 결과에서 어디에 위치하는지에 따라 이 루트 노드의 왼쪽에 오는 자손들과
오른쪽에 오는 자손들이 누군지 알 수 있다.
  * 따라서 주어진 결과들로 왼쪽 자손들의 전위 순회 결과와 중위 순회 결과, 오른쪽 자손들의 전위 순회 결과와 중위 순회 결과를
를 구할 수 있고, 이를 재귀 호출하여 원본을 복원할 수 있다.

* 시간 복잡도 :
  * $O(n^2)$
