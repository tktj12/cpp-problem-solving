# Problem
* https://algospot.com/judge/problem/read/TREASURE
* 단순 다각형과 직사각형이 겹치는 부분의 넓이 구하기
* 기하 문제
* 난이도 : 상

# Solution

### 방법 1 : 서덜랜드-호지맨 알고리즘
* 풀이
  * 단순 다각형을 통과하는 직선의 한쪽 평면(반평면)과 단순 다각형의 교집합을 계산한다. 
즉, 단순 다각형을 직선으로 자른 한쪽을 계산한다.
  * 직선을 기준으로 왼쪽 혹은 오른쪽에 있는 점들과 직선과 교차하는 점들을 포함하는 새 단순 다각형을 계산한다.
  * 이렇게 네 개의 직선에 대해 계산한다.

* 시간 복잡도 :
  * $O(n)$
