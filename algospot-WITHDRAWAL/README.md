# Problem
* https://algospot.com/judge/problem/read/WITHDRAWAL
* 적절하게 수강 취소하여 평균 성적 올리기
* 난이도 : 중

# Solution

### 방법 1 : 그리디
* 풀이
  * 모든 과목을 포함한 경우부터 시작
  * 여기서 과목 하나를 빼면 누적 등수를 줄일 수 있는지, 줄이면 최대 얼마나 줄일 수 있는지 판단한다.
  * 최대한 줄일 수 있는 과목을 선택하여 제거한다.
  * 가능한 한 과목을 하나씩 계속 빼가면서 누적 등수를 계속 줄여나간다.

* 시간 복잡도 :
  * $O(N^2)$
